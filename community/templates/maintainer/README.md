# Maintainer Template

Codebase steward for existing projects. Focuses on test health, documentation upkeep, technical debt tracking, and safe incremental improvements.

## Install

```bash
soma install template maintainer
```

## What You Get

- **5 muscles** — test-hygiene (hot), doc-hygiene (hot), safe-file-ops, incremental-refactor, task-tooling
- **6 protocols** — tool-discipline, quality-standards, working-style, pre-flight, frontmatter-standard, task-tracking
- **Identity** — maintenance-focused working style
- **Settings** — lower hot threshold (6), higher protocol load for stewardship

## Best For

- Inherited codebases that need cleanup
- Keeping test suites healthy after refactors
- Documentation maintenance and plan hygiene
- Technical debt reduction
- Pre-release audits and health checks
