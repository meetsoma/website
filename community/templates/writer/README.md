# Writer

A writing-focused agent for technical content, blogging, and documentation. Accuracy first, honest voice.

## Install

```bash
soma install template writer
```

## What You Get

- **3 muscles** — precision-edit (hot), doc-hygiene, micro-exhale
- **4 protocols** — working-style, frontmatter-standard, correction-capture, detection-triggers
- **SOMA.md** — writing-focused identity tuned for accuracy and voice consistency
- **Settings** — higher muscle budget (4000 tokens), relaxed heat thresholds for writing workflows

## Recommended Add-ons

These hub AMPS pair well with writing work:

- **scope-discipline** — stay focused on what was asked
- **breath-cycle** — manage context across long writing sessions

```bash
soma install muscle scope-discipline
```

## Best For

- Dev blogs and building-in-public content
- Technical documentation
- Co-authored posts (human + agent)
- Content that needs fact-checking against code
