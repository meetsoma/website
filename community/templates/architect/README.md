# Architect

A structure-focused agent for engineering teams. Thinks about code organization before writing code. Reviews diffs for standards compliance. Documents decisions, not just implementations.

## Install

```bash
soma install template architect
```

## What You Get

- **4 muscles** — precision-edit (hot), incremental-refactor, task-tooling, doc-hygiene
- **8 protocols** — tool-discipline, working-style, quality-standards, git-identity, frontmatter-standard, pre-flight, pattern-evolution, correction-capture
- **SOMA.md** — structure-focused identity that prioritizes consistency over speed
- **Settings** — lower warm threshold (2), faster protocol activation for architecture work

## Recommended Add-ons

These hub AMPS pair well with architecture work:

- **ripple-awareness** — map blast radius before committing changes
- **plan-lifecycle** — end-to-end flow from idea to shipped
- **maps** — read the MAP before starting any complex task
- **plan-hygiene** — keep plans current, archive what's done

```bash
soma install muscle ripple-awareness
```

## Best For

- Teams that care about code structure and documentation
- Projects with multiple contributors
- Repos that need consistent commit attribution
- Engineering orgs building internal standards
